Система динамического мониторинга товаров
Инструмент для анализа показателей товаров на маркетплейсе Wildberries. Позволяет отслеживать остатки, заказы и логистику по отчетам с интерактивной визуализацией.



Основные возможности

Интерактивные графики: Визуализация метрик (заказы, остатки, товары в пути) с использованием Matplotlib.

Умная легенда: Поддержка фильтрации конкретных товаров на графике в один клик. При нажатии отображаются точные числовые показатели.

Адаптивный интерфейс: Программа автоматически подстраивается под разрешение экрана (от 1366x768 до 1920x1080), используя режимы умного масштабирования.

Сравнение товаров: Отдельный модуль для сопоставления динамики нескольких артикулов на одном холсте.

Локальное хранение: Автоматическое сохранение отчетов в папку пользователя для удобной работы.



Технологический стек

Язык: Python

Интерфейс: CustomTkinter

Анализ данных: Pandas

Визуализация: Matplotlib
